LATIHAN SOAL - SETS & DICTIONARIES
===================================

CARA JALANKAN:
python latihan_soal.py


ISI FILE:
---------
File ini berisi jawaban 5 soal latihan:

1. Deduplikasi
   - Hapus duplikat dari list
   - Pertahankan urutan pertama

2. Intersection Dua Array
   - Cari elemen yang ada di kedua list
   - Gunakan set untuk efisiensi

3. Anagram Check
   - Cek apakah dua string adalah anagram
   - Pakai dictionary untuk hitung frekuensi

4. First Recurring Character
   - Temukan karakter pertama yang muncul lagi
   - Pakai set untuk tracking

5. Simulasi Buku Telepon
   - Program interaktif dengan menu
   - Tambah, cari, tampilkan, keluar


CARA KERJA:
-----------
Saat dijalankan:
- Soal 1-4 akan otomatis testing dan print hasilnya
- Soal 5 akan minta input dari user (menu interaktif)


STRUKTUR DATA YANG DIPAKAI:
----------------------------
- Set: untuk cek keanggotaan cepat
- Dictionary: untuk pasangan key-value
- List: untuk hasil yang terurut
