GAME OF LIFE
============

CARA JALANKAN:
python game_of_life.py


CARA PAKAI:
- Klik kotak = hidupin/matikin
- START = mulai
- STOP = berhenti  
- RESET = hapus semua
- RANDOM = isi acak


FILE:
game_of_life.py  -> program
penjelasan.txt   -> penjelasan
README.txt       -> ini


KALAU ERROR:
Pastikan Python sudah terinstall.
Tkinter biasanya udah ada, gak perlu install.
