PENJELASAN GAME OF LIFE
========================

1. APA ITU GAME OF LIFE?
-------------------------
Game of Life adalah simulasi yang dibuat oleh John Conway tahun 1970.
Ini bukan game yang dimainkan, tapi simulasi otomatis.

Ada grid (seperti papan catur) yang isinya kotak-kotak.
Setiap kotak = 1 sel.
Setiap sel bisa hidup (warna hijau) atau mati (warna hitam).


2. ATURANNYA APA?
-----------------
Ada 4 aturan sederhana:

ATURAN 1: Kalau sel hidup punya kurang dari 2 tetangga hidup
            → Sel mati (kesepian)

ATURAN 2: Kalau sel hidup punya 2 atau 3 tetangga hidup
            → Sel tetap hidup

ATURAN 3: Kalau sel hidup punya lebih dari 3 tetangga hidup
            → Sel mati (terlalu sesak)

ATURAN 4: Kalau sel mati punya tepat 3 tetangga hidup
            → Sel jadi hidup (lahir)

Tetangga = 8 kotak di sekitar sel (atas, bawah, kiri, kanan, diagonal)


3. STRUKTUR DATA YANG DIPAKAI
------------------------------
Kita pakai ARRAY 2D (list of lists) untuk simpan grid.

Contoh grid 3x3:
    grid = [
        [0, 1, 0],
        [1, 1, 1],
        [0, 1, 0]
    ]

0 = sel mati
1 = sel hidup

Cara akses: grid[baris][kolom]
Misalnya grid[0][1] = 1 (baris 0, kolom 1, isinya hidup)


4. CARA KERJA ALGORITMA
-----------------------
Setiap generasi:

1. Buat grid baru yang kosong
2. Loop semua sel di grid lama
3. Untuk tiap sel:
   - Hitung ada berapa tetangga hidup
   - Cek aturan 1-4
   - Update grid baru
4. Grid baru jadi grid sekarang

Kenapa harus bikin grid baru?
Supaya semua sel diupdate BERSAMAAN, bukan satu-satu.


5. FUNGSI-FUNGSI PENTING
------------------------

buat_grid_baru()
  → Buat grid kosong berisi 0 semua

hitung_tetangga(grid, baris, kolom)
  → Hitung berapa tetangga hidup di sekitar sel
  → Loop 8 arah pakai for i in range(-1, 2)
  → Pakai modulo (%) biar grid melingkar

generasi_berikutnya(grid_lama)
  → Buat grid baru
  → Loop semua sel
  → Terapkan 4 aturan
  → Return grid baru


6. CARA PAKAI PROGRAM
---------------------
1. Jalankan: python game_of_life.py
2. Klik kotak untuk hidupin/matikin sel
3. Tekan START untuk mulai simulasi
4. Tekan STOP untuk berhenti
5. Tekan RESET untuk bersihkan semua
6. Tekan RANDOM untuk isi acak


7. KENAPA MENARIK?
------------------
Dari 4 aturan sederhana, muncul pola-pola yang keren:

- Ada pola yang diam aja (gak berubah)
- Ada pola yang berkedip-kedip
- Ada pola yang bergerak!
- Ada pola yang terus menghasilkan pola baru

Ini namanya "emergent behavior" - perilaku kompleks dari aturan sederhana.
