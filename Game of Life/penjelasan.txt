PENJELASAN GAME OF LIFE
========================

APA ITU?
--------
Game of Life itu simulasi otomatis. Ada kotak-kotak (grid), 
setiap kotak bisa hidup (hijau) atau mati (hitam).

Tiap kotak punya 8 tetangga (atas, bawah, kiri, kanan, diagonal).


ATURANNYA?
----------
Ada 4 aturan sederhana:

1. Kotak hidup + tetangga kurang dari 2 = mati
2. Kotak hidup + tetangga 2 atau 3 = tetap hidup  
3. Kotak hidup + tetangga lebih dari 3 = mati
4. Kotak mati + tetangga tepat 3 = jadi hidup


CARA PAKAI PROGRAM?
-------------------
1. Jalankan: python game_of_life.py
2. Klik kotak untuk hidupin/matikin
3. Tekan START untuk mulai
4. Tekan STOP untuk berhenti
5. Tekan RESET untuk hapus semua
6. Tekan RANDOM untuk isi acak


CARA KERJANYA?
--------------
1. Ada variabel "grid" yang isinya list 2D
   - grid[baris][kolom] = 0 (mati) atau 1 (hidup)

2. Fungsi hitung_tetangga():
   - Cek 8 kotak di sekitar
   - Hitung ada berapa yang hidup

3. Fungsi update_grid():
   - Buat grid baru
   - Loop semua kotak
   - Hitung tetangga tiap kotak
   - Pakai aturan 1-4 untuk isi grid baru
   - Grid lama jadi grid baru

4. Fungsi gambar_grid():
   - Loop semua kotak
   - Gambar hijau kalau 1, hitam kalau 0


KENAPA PAKAI LIST 2D?
---------------------
Biar gampang akses kotak pakai grid[baris][kolom].
Kayak Excel, ada baris dan kolom.


APA ITU % (MODULO)?
-------------------
Biar grid melingkar. Kalau sampai ujung kanan, 
balik ke kiri lagi.

Contoh:
(40 + 1) % 40 = 1 % 40 = 1
(0 - 1) % 40 = -1 % 40 = 39


YANG PENTING DIPAHAMI:
----------------------
- Grid = list 2D untuk simpan data
- 0 = mati, 1 = hidup
- Loop for untuk cek semua kotak
- Hitung tetangga pakai loop 8 arah
- Buat grid baru biar update serentak
